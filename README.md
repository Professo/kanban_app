# Kanban App

A project management app like trello made using WebPacks and ReactJs.

Demo: http://perminder-klair.github.io/kanban_app

## Development

```
npm start
```

## Build for production

```
npm start build
```

## Deploy to Github Pages

```
npm start deploy
```

## Check build stats

```
npm start stats
```
